# Return Project

서원대학교 비교과 프로그램 **리턴(Re-Turn) 프로젝트** 참여 자료를 정리한 저장소입니다.  
메타리터러시 수업 자료, 워크시트, 최종 소감문을 보관합니다.

---

## ✨ 핵심 내용

| 분류 | 내용 |
| --- | --- |
| 프로그램 | 리턴(Re-Turn) 프로젝트 |
| 주제 | 메타리터러시, 학습 성찰, 비교과 활동 정리 |
| 자료 형태 | PPTX, HWPX, HWP |
| 목적 | 참여 과정 기록과 최종 소감문 보관 |

## 📝 정리 기준

활동 과정에서 작성한 문서와 발표 자료를 원본 형식 그대로 보관합니다.  
파일명에는 과제명과 학번/이름을 유지해 제출 자료와 구분할 수 있도록 했습니다.
